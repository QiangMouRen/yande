# 获取yande.re图片

example.php 为示例文件，作为参考
<pre>
获取首页的图片 ./example.php&page=页数
获取tag的图片 ./example.php?tag=tag名称&page=页数
搜索tag ./example.php?do=searchTag&keyword=关键词&order=排序方式&type=搜索类型
排序方式 name => 名称
				count => tag下图片数量
				date => 时间
搜索类型 0 => General
				1 => Artist
				3 => Copyright
				4 => Character
				5 => Circle
				6 => Faults

</pre>
